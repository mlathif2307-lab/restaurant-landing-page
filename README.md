# 🌶️ Sambel Dadak - Website Rumah Makan Nusantara

Sebuah platform landing page interaktif untuk restoran "Sambel Dadak / Rumah Makan Nusantara" yang dirancang menggunakan teknologi web fundamental (HTML, CSS, dan JavaScript). Project ini dilengkapi dengan sistem pemesanan menu dan simulasi pembayaran dinamis secara *real-time*.

## 🚀 Fitur Utama

- **Responsive Landing Page:** Tampilan modern dan elegan yang dioptimalkan menggunakan font Google (Montserrat & Poppins) serta ikon Font Awesome.
- **Interactive Menu Ordering System:** Fitur modal pop-up yang memungkinkan pengguna memilih kuantitas makanan, minuman, hingga dessert secara interaktif.
- **Real-time Bill Calculation:** Menggunakan JavaScript untuk menghitung subtotal dan grand total pembayaran secara otomatis saat kuantitas menu diubah.
- **Multi-method Payment Gateway Simulation:** Form pembayaran dinamis yang mendukung berbagai opsi metode pembayaran (Kartu Kredit, E-Wallet, Transfer Bank, dan Tunai) lengkap dengan validasi form input.
- **Smooth Navigation Scrolling:** Navigasi antar bagian halaman (Home, About, Makanan, dll) yang halus demi kenyamanan pengalaman pengguna (UX).

## 🛠️ Tech Stack

- **Front-End:** HTML5, CSS3 (Custom Variables & Linear Gradients), JavaScript (Vanilla JS)
- **Library & Fonts:** Font Awesome Icons, Google Fonts (Poppins, Montserrat)

## 📂 Struktur File

- `index.html` - Struktur utama halaman dan modal pemesanan.
- `style.css` - Desain visual, layouting (Flexbox/Grid), dan tema warna beraksen merah-emas.
- `script.js` - Logika interaksi modal, kalkulasi total belanja, dan penanganan form pembayaran.
